Experiment 4 - Infinite Worlds - River Flow

Description

I used the infinite extent of my game to represent an endlessly running river. The state of the water is always changing. As the user moves through the world they will see various areas that have different water movements with random highlights as if the sun is shining upon the river. Different worlds will have different highlight patterns with slightly different water movements. When the use clicks on a tile, a small ripple will be made and will eventually fade away.

Technical

This project uses procedural generation techniques to generate the everflowing river. It utilizes a hashing function to determine which tiles are highlighted.
