Experiment 4 - Infinite Worlds - Dance Floor

Description

I used the infinite extenet of my world to represent an endless dance floor with endless possibilites of expression. As the user moves though the world, they will see varying color changing areas with white patches scattered through out to represent spotlights. Different worlds will have different spotlight light arrangements and there will be variations with the rainbow lights as well. Clicking on a tile will disable it and fill it with a black color.

Technical

This project uses prodedural generation techniques to create a constantly changing color landscape that has random patches of light.
